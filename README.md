# RESTful Student Management Backend

## Project Description
This project focuses on the design and development of a RESTful backend system for managing student information. The system follows resource-oriented architecture and correct usage of HTTP methods to perform CRUD operations on student resources.

## Objectives
- To design RESTful APIs using resource-oriented principles
- To demonstrate correct usage of HTTP methods (GET, POST, PUT, DELETE)
- To build an industry-relevant backend system

## Technologies (Planned)
- Java
- Spring Boot
- RESTful Web Services
- JSON
- Postman

## Core Features
- Add a student
- View all students
- Update student details
- Delete student records

## Project Status
Initial setup completed. API design and implementation in progress.
